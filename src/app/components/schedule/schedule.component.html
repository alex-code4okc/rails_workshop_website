Schedule

The event is broken up into two parts: an install fest and the workshop itself.

Install Fest

Friday, March 29th, 2019 from 5:30pm - 7:30pm

Completely optional, this is to give attendees a head start on setting up their computers for the main event on Saturday, the 30th.

We'll make sure that all of the tools needed to follow the curriculum are up and running so you can dive right in. Can't make it? No worries! There is plenty of time on Saturday to get set up.

Workshop

Saturday, March 30th, 2019 from 9:30am - 5:30pm

This is the main event! We'll break into small groups, and coaches will help you dive into the curriculum. Food is provided, and we'll have frequent breaks to make sure that your brain stays fresh. We'll also have talks from people in the local community.
