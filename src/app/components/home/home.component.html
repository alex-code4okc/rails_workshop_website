Welcome!

Women are underrepresented in the tech market. We want to change that. The Rails OKC Workshop is an event for women attendees only that serves as an introduction to web development. Our goal is to provide a supportive, welcoming environment that empowers women to become developers, designers, engineers, or simply spark a new hobby. Using the powerful Ruby on Rails framework, attendees will learn how to build their very own websites. We hope that you will join us.
